# SmartBudget

A personal finance tracker that is tailored to young adults.

## What it does

This app lets you add transactions (income or expenses), set savings goals, and see visualizations of your spending.

## How to run it

1. Install dependencies:
```
npm install
cd server && npm install
cd ../client && npm install
cd ..
```

2. Start the website:
```
npm run dev
```

3. Open the web browser and view it to http://localhost:5173
